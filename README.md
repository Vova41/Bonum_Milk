# NeMilk — мониторинг температуры молока

Веб-приложение для контроля температуры молока во время транспортировки. Два интерфейса: водитель видит температуру в реального времени и прогноз, диспетчер наблюдает за маршрутом и общается с водителем через чат.

## Роли

- **Водитель** (`/driver`) — текущая температура груза, история за смену, прогноз погоды и рисков, чат с диспетчером
- **Диспетчер** (`/dispatcher`) — график температуры, статус доставки, чат с водителем

## Стек

- Next.js 16 / React 19
- Tailwind CSS 4
- TypeScript

## Запуск

```bash
npm install
npm run dev       # разработка на localhost:3000
npm run build     # production сборка
npm run start     # запуск production сборки
```

## Деплой

Проект задеплоен на Vercel: https://nemilk.vercel.app

Для обновления:
```bash
vercel --prod
```
